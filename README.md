# Rainbow CSS

Windows向けプロフェッショナルUIデザインCSSフレームワーク。

## 特長
- ブランドカラーによる美しいテーマ
- ボタン・フォーム・リスト・通知などのUIコンポーネント
- アクセシビリティ＆レスポンシブ対応
- Windows 10/11最適化

## 使い方
1. `rainbow.css` をHTMLに読み込む
2. クラスを付与して利用

```html
<link rel="stylesheet" href="rainbow.css">
<button class="btn btn-primary">Primary</button>
```
